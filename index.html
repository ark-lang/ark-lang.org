About Alloy

Alloy is a systems programming language. It started off as a small, experimental side-project of mine to learn more about compilers. My original goal with Alloy is to make a cleaner C without deviating from C's original goals of simplicity.

Goals

  • Fast
  • Multi-Paradigm
  • Strongly Typed
  • Concise, clean, and semantic syntax
  • No garbage collection

How it looks

// binding for c's printf
