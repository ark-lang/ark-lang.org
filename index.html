Ark is a systems programming language.

Contribute

Ark is open source, and can be found on GitHub! Send us a pull request, comment on some existing issues, propose a feature, or find some bugs and report your findings in the issue handler! You can also find many resources on our organisation page, such as the Documentation, and various examples & syntax highlighting for your editor.

Why?

Our goal is to write a modern systems programming language that adopts many modern concepts and advancements in the field of compilers and language design over the past years.

We want a language that has just enough features for programming systems, yet not too many that the language is hard to learn.

Example

Here's a small program written in Ark. 

// binding for c's printf
[c] func printf(format: str, ...): int;

func main(): int {
	printf("Hello, World\n");
	return 0;
}
