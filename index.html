About Ark

Compilers, compiler design and programming language design have been vastly improved since languages such as C were first invented, as has the software that we write in these languages. Many new concepts have been added to languages like C++ and C, yet they have maintained backwards compatibility, including the flaws of these languages.

Ark exists to include all of these modern features, and improvements that have occured in the last years; yet exclude all of the ugly flaws present in languages like C and C++.

Goals

  • Fast
  • Multi-Paradigm
  • Strongly Typed
  • Concise, clean, and semantic syntax
  • No garbage collection
